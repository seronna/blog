# 使用飞书多维表格自动阅读文献并生成周报

## 功能介绍
本项目基于飞书多维表格，实现了自动阅读文献并生成周报的功能。用户可以在飞书多维表格中上传文献的pdf文件或者输入文献链接，项目会自动读取文献内容，并通过自动化程序每周定时生成周报。

## 主要流程
首先保存这个飞书多维表格的副本，这里是总的面板，有总的文献列表、每周的周报列表、以及一些配置项。然后你可以在这里点击上传pdf文件或者输入文献链接，项目会自动读取文献内容。，或者你也可以在这里上传，把这个链接分享到你的QQ、或者飞书等地方都可以。上传之后就会自动生成这些、比如文章标题、摘要等等、同时你可以在这里修改你要用的ai模型（有一些需要去火山引擎配置一下）、同时这里也可定义自定义的指令。然后后面这些都是ai生成的。然后还有一个就是工作流，我这里设置每周五定时出触发、当他触发时就去查找这个表上传日期晚于今天的前七天的日期（这里你也可以自己设置想要的）然后查找全文总结（这里你也可以改自己想要的）然后ai生成文本，这是我的一个自定义指令，然后选一个人发送，整个流程就结束了，更多的可以自己设置。
